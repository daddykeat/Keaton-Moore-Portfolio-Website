# Keaton-Moore-Portfolio-Website

Personal website and portfolio for Keaton Moore (Strata Dev), featuring web development work, creative design projects, and motion graphics case studies.

## Updates & Changes

### Site Structure Updates
- Built a full single-page portfolio layout with dedicated sections for About, Services, Skills, Web Development Projects, Creative Work, Motion Workflow, Experience, and Contact.
- Added internal anchor-based navigation with a sticky header for quick section-to-section browsing.
- Added a skip-to-content link to improve keyboard accessibility.

### UI/UX and Visual Updates
- Implemented a modern visual style using layered gradients, glass-style panels, shadows, and accent color tokens in CSS variables.
- Added a responsive layout system for desktop and mobile viewing.
- Added reveal-on-scroll animations for staged section entrance.
- Added subtle parallax background movement for additional depth.

### Interactivity and Behavior Updates
- Added a mobile navigation toggle with click-outside close behavior.
- Added nav auto-close behavior after link selection on mobile.
- Implemented scrollspy logic to highlight the active navigation link based on viewport position.
- Added dark/light theme toggle with localStorage persistence.
- Added dynamic footer year rendering via JavaScript.

### Content and Portfolio Updates
- Expanded hero content to clearly communicate services, strengths, and brand identity (Strata Dev).
- Added service cards for custom websites, landing pages, updates/bug fixes, and interactive web tools.
- Added web project showcase cards with external live demo links.
- Added creative work showcase with asset previews and direct image links.
- Added motion workflow previews with embedded video cards and links to detailed process pages.

### Motion Project Detail Pages Added
- Added `projects/level-up.html` with creative brief, storyboard, assets, techniques, and process notes.
- Added `projects/snack-time.html` with project intent, animation principles, techniques, and workflow breakdown.
- Added `projects/bourbon-street-awakening.html` with concept summary, storyboard beats, palette, and production process.

### Tech Stack and Dependencies
- Core stack: HTML, CSS, and vanilla JavaScript.
- External dependencies: Google Fonts (Manrope) and Font Awesome icons via CDN.
- Media assets stored in the `assets/` directory and reused across homepage/project pages.
